# 职业密码 - MBTI职业性格测试

[🚀 在线体验](https://zimai233.github.io/mbti-career-test/)

🔮 一个精美的MBTI职业性格测试网页应用，通过30道题目分析你的性格特质，匹配最适合你的职业方向。

## ✨ 功能特性

- **30道精心设计的题目** - 涵盖10个性格维度
- **10个评估维度** - 社交倾向、信息获取、决策方式、生活方式、领导力、创意性、独立性、压力承受、执行力、沟通风格
- **12种职业类型匹配** - 领袖型、创意型、技术型、分析型、服务型、艺术型、商业型、执行型、研究型、社交型、操作型、管理型
- **个性化匹配度分析** - 每个职业都有百分比匹配度展示
- **雷达图可视化** - 直观展示你的性格特质分布
- **MBTI人格分析** - 获得你的16型人格代码和描述
- **进度保存** - 自动保存答题进度，不用担心中途退出丢失数据

## 🛠️ 技术栈

- React 18 + TypeScript
- Vite
- Tailwind CSS
- Recharts (雷达图)

## 🚀 在线体验

直接打开 `dist/index.html` 文件即可使用，或部署到任何静态托管服务。

## 📦 本地运行

```bash
# 安装依赖
npm install

# 开发模式
npm run dev

# 构建生产版本
npm run build

# 预览生产版本
npm run preview
```

## 🎨 设计理念

采用神秘的宇宙主题配色，配合流畅的动画效果，打造沉浸式的测试体验：
- 深邃的宇宙蓝紫色渐变背景
- 金色和紫色的点缀色彩
- 优雅的衬线体标题 + 无衬线正文
- 丰富的微交互动画
